This project contains the web application code for the Crispr / Cas9 project


## Installation

- Clone this repo:

```bash
$ git clone https://github.com/emepyc/project-score
$ cd project-score 
```

- Install all dependencies:

```shell
$ yarn install
```

- Start the application

```
$ yarn start
```

- To build the project

```
$ yarn build
```
